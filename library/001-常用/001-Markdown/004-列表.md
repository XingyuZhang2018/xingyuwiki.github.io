## <font color="#FF4500">[列表]</font><br />
#### 无序列表
使用 * + - 来做为列表的项目标记

```markdown
* Candy.
* Gum.
+ Booze.
- Booze.
  + 嵌套
  * 嵌套
```

* Candy.
* Gum.
+ Booze.
- Booze.
  + 嵌套
  * 嵌套

#### 有序列表
一般的数字接着一个英文句点作为项目标记，效果与数字无关只与顺序有关

```markdown
1. Red
2.  Green
3.    Blue
```

1. Red
2.  Green
3.    Blue
